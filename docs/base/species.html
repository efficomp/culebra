base.Species class¶

class Species¶

Base class for all the species in culebra.

Each individual evolved within a wrapper must belong to a species which constraints the individual parameter values.

Methods¶

abstract Species.is_member(ind: base.abc.Individual) → bool¶

Check if an individual meets the constraints imposed by the species.

This method must be overriden by subclasses to return a correct value.

Parameters

ind (Individual) – The individual

Returns

True if the individual belongs to the species, or False otherwise

Return type

bool
