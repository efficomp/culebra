culebra.solution module¶

Solution module.

This module provides several solution and species implementations for several problems. Currently, the following sub-modules are available:

  • abc: Abstract base classes needed for each type of metaheuristic developed as a culebra’s Trainer

  • feature_selection: Solutions and species defined for feature-selection problems.

  • parameter_optimization: Solutions and species targeted for parameter optimization problems.

  • tsp: Solutions and species for the traveling salesman problem.
