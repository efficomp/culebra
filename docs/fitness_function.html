fitness_function module¶

Implementation of different fitness functions.

This module provides fitness functions for several problems. They are grouped within the following sub-modules:

  • feature_selection sub-module: Fitness functions to select a minimal subset of features from a Dataset.

  • classifier_optimization sub-module: Fitness functions to find the optimal hyper-parameters of the classifier used within a Wrapper procedure.

  • cooperative sub-module: Fitness functions to solve the two former problems in a cooperative way.
