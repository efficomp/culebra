wrapper.single_pop module¶

Implementation of single-population generational evolutionary algorithms.

This module provides the following wrapper method implementations:

  • SinglePop: Abstract base class for all the single-population generational wrappers

  • Evolutionary: Simple evolutionary algorithm

  • Elitist: Elitist evolutionary algorithm

  • NSGA: Multi-objective evolutionary algorithm, based on Non-dominated sorting, able to run the NSGA-II or the NSGA-III algorithms

Attributes¶

Default population size.
