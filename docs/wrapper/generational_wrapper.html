wrapper.generational_wrapper¶

Provides the GenerationalWrapper class.

This class is the base class for all the generational wrapper procedures developed within the Wrapper subpackage. It is an abstract class. The _do_generation() method must be implemented in its subclasses to obtain a complete generational algorithm.
