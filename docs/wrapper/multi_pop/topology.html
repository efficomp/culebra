wrapper.multi_pop.topology module¶

Functions¶

Return the destinations reachable from origin.

Parameters

  • origin (int) – The index of the origin subpopulation wrapper

  • num_subpops (int) – The number of subpopulations

  • offset (int, optional) – Offset applied to origin, defaults to DEFAULT_RING_OFFSET

Returns
