wrapper.nsga_wrapper¶

Provides the NSGAWrapper class.

This class allows to run the NSGA2 or NSGA3 algorithm as the search method within the wrapper method.

Bases: culebra.wrapper.evolutionary_wrapper.EvolutionaryWrapper

NSGA-based wrapper method.

__init__(individual_cls, species, **params)¶

Initialize the wrapper method.

Parameters

  • individual_cls (Any subclass of Individual) – Individual representation.

  • species (Species) – The species the individual will belong to

  • pop_size (int, optional) – Population size, defaults to DEFAULT_POP_SIZE

  • n_gens (int, optional) – Number of generations, defaults to DEFAULT_N_GENS

  • xover_func (Any callable object, optional) – Crossover function, defaults to the crossover() method of individual_cls

  • xover_pb (float, optional) – Crossover rate, defaults to DEFAULT_XOVER_PB

  • mut_func (Any callable object, optional) – Mutation function, defaults to the mutate() method of individual_cls

  • mut_pb (float, optional) – Mutation rate, defaults to DEFAULT_MUT_PB

  • mut_ind_pb (float, optional) – Independent gene mutation probability, defaults to DEFAULT_MUT_IND_PB

  • sel_func (Any callable object, optional) – Selection function (selNSGA2() or selNSGA3()), defaults to DEFAULT_SEL_FUNC

  • sel_func_params (dict, optional) – Selection function parameters. If NSGA-III is used, this attribute must include a key named ‘ref_points’ containing a dict with the parameters needed to generate the reference points (the arguments of uniform_reference_points()). Since sel_func defaults to NSGA-II, the default value for sel_func_params is DEFAULT_SEL_FUNC_PARAMS

  • checkpoint_freq (int, optional) – Frequency for checkpointing, defaults to DEFAULT_CHECKPOINT_FREQ

  • checkpoint_file (str, optional) – File path for checkpointing, defaults to DEFAULT_CHECKPOINT_FILE

  • random_seed (int, optional) – Random seed for the random generator, defaults to None

  • verbose (bool) – Whether or not to log the statistics, defaults to __debug__

Raises

TypeError – If any parameter has a wrong type
