Tools subpackage¶

This package provides several tools:

  • Since individuals are generated and modified randomly, it is necessary an Individual tester to test extensively the random generation, crossover and mutation of all the Individual implementations.

  • Some Feature metrics are also necessary to estimate the relevance of the input features from the Wrapper results

  • Automated experimentation is also a quite valuable characteristic when a Wrapper method has to be run many times. Culebra provides this features by means of a TOML configuration file and the config_manager module, which is able to generate experiments and even batchs of experiments from a single configuration file
