wrapper module¶

Implementation of some wrapper algorithms.

By the moment, culebra is centered on the use of generationsl evolutionary algorithms as the search engine for wrapper procedures. Thus, the wrapper module is composed by:

  • The Generational class, which inherits from Wrapper, and provides the interface for all the evolutionary wrappers within this module.

  • The single_pop sub-module, which provides single-population generational wrappers.

  • The multi_pop sub-module, which provides multiple-population generational wrappers.

Attributes¶

DEFAULT_STATS_NAMES = ('Gen', 'Pop', 'NEvals')¶

Default statistics calculated for each generation.

DEFAULT_NUM_GENS = 100¶

Default number of generations.
