Wrapper subpackage¶

This package provides an abstract base class for all the generarational wrapper procedures:

  • wrapper.generational_wrapper

And two generational wrapper procedures: a simple evolutionary algorithm and another one, based on Non-dominated sorting, able to run NSGA-II and NSGA-III algorithms:

  • wrapper.evolutionary_wrapper

  • wrapper.nsga_wrapper
