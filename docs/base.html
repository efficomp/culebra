base module¶

Base classes of culebra.

Culebra is based in some base classes to define the fundamental pieces that are necessary to solve a feature selection problem. The base module defines:

  • A Base class from which all the classes of culebra inherit

  • A Dataset class containing samples, where the most relevant input features need to be selected

  • An Individual class, which will be used within the Wrapper class to search the best subset of features

  • A Species class to define the characteristics of the individuals

  • A FitnessFunction class to evaluate the individuals during the search process

  • A Fitness class to store the fitness values for each Individual

  • A Wrapper class to find the most representative features of the Dataset

  • A checker module with functions to verify different kinds of objects.

Attributes¶

DEFAULT_SEP = '\\s+'¶

Default column separator used within dataset files.

DEFAULT_STATS_NAMES = ('Iter', 'NEvals')¶

Default statistics calculated for each iteration of the wrapper.

DEFAULT_OBJECTIVE_STATS = {'Avg': <function mean>, 'Max': <function amax>, 'Min': <function amin>, 'Std': <function std>}¶

Default statistics calculated for each objective within a wrapper.

DEFAULT_CHECKPOINT_ENABLE = True¶

Default checkpoinitng enablement for wrappers.

DEFAULT_CHECKPOINT_FREQ = 10¶

Default checkpointing frequency for wrappers.

DEFAULT_CHECKPOINT_FILENAME = 'checkpoint.gz'¶

Default checkpointing file name for wrappers.

DEFAULT_VERBOSITY = True¶

Default verbosity for wrappers.

DEFAULT_INDEX = 0¶

Default wrapper index. Only used within distributed wrappers.

DEFAULT_CLASSIFIER = <class 'sklearn.naive_bayes.GaussianNB'>¶

Default classifier for fitness functions.
