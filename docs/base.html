Base subpackage¶

Culebra is based in some base modules to define the fundamental pieces that are necessary to solve a feature selection problem. The base subpackage defines:

  • A dataset with samples, where the most relevant input features need to be selected

  • The individuals, which will be used within the wrapper method to search the best subset of features

  • A species to define the characteristics of the individuals

  • The fitness function to guide the search towards optimal solutions

  • The wrapper method to perform the search
