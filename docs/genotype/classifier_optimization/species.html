genotype.classifier_optimization.Species class¶

class Species(lower_bounds: Sequence[int | float], upper_bounds: Sequence[int | float], types: Optional[Sequence[Union[Type[int], Type[float]]]] = None, names: Optional[Sequence[str]] = None)¶

Create a new species.

Parameters:

  • lower_bounds (Sequence of int or float values) – Lower bound for each hyperparameter

  • upper_bounds (Sequence of int or float values) – Upper bound for each hyperparameter

  • types (Sequence of type (int or float), optional) – Type of each hyperparameter. All the hyperparameters will be treated as float if omitted. Defaults to None

  • names (Sequence of str, optional) – Name of each hyperparameter. Defaults to None

Raises:

  • TypeError – If any of the attributes is not a Sequence

  • ValueError – If the sequences have different lengths

  • ValueError – If the sequences are empty

  • ValueError – If the type of any bound does not match with its corresponding type in types

  • ValueError – If any lower bound is greater to or equal than its corresponding upper bound

  • ValueError – If any name is not an instance of str

  • ValueError – If there is any repeated name
