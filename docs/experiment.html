experiment module¶

Automated experimentation is also a quite valuable characteristic when a Wrapper method has to be run many times. Culebra provides this features by means of a TOML configuration file and the ConfigManager class, which is able to generate experiments and even batchs of experiments from a single configuration file.

This module provides some classes to make the experimentation easier.

  • ConfigManager class: Build an Experiment or a Batch of experiment from a TOML configuration file.

  • Base class: Base class for Experiment and Batch

  • Experiment class: Run an experiment

  • Batch class: Run a batch of experiments
