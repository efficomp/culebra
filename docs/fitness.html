fitness module¶

Fitness classes for feature selection problems.

This module provides several fitness classes designed to solve feature selection problems using a wrapper method:

  • NumFeatsFitness: Dummy single-objective fitness class that tries to minimize the number of features selected by a wrapper.

  • KappaIndexFitness: Single-objective fitness class that maximizes the Kohen’s Kappa index of the solutions found by a wrapper. The Kappa index may be claculated over the whole traininf dataset or only over a validation proportion of the data.

  • KappaNumFeatsFitness: Bi-objective fitness class that combine the two above fitness classes.

__init__(**params)¶

Create the fitness.

If valid_prop is not None, the classifier is trained with (1 - valid_prop) proportion of the samples and validated with the remaining samples. Otherwise the classifier is trained and tested with all the available data.

Parameters

  • valid_prop (A real number, optional) – Validation proportion. If not None, the classifier is trained with (1 - valid_prop) proportion of the samples and validated with the remaining samples. Otherwise the classifier is trained and tested with all the available data. Defaults to None.

  • classifier (Any subclass of ClassifierMixin) – The classifier to be used within this wrapper.

  • thresholds (float or a Sequence of float numbers, optional) – Thresholds to assume if two fitness values are equivalent. If only a single value is provided, the same threshold will be used for all the objectives. A different threshold can be provided for each objective with a Sequence. Defaults to DEFAULT_FITNESS_THRESHOLD

Raises

  • TypeError – If valid_prop is not a real number

  • ValueError – If valid_prop is not in (0, 1)

  • TypeError – If classifier is not a valid classifier

  • TypeError – If thresholds is not a float value or a Sequence of float numbers

  • ValueError – If a negative threshold is provided

Fitness objects are compared lexicographically. The comparison applies a similarity threshold to assume that two fitness values are similar (if their difference is lower then the similarity threshold).
