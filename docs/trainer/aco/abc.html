culebra.trainer.aco.abc module¶

Abstract base classes for different ACO-based trainers.

This module provides several abstract classes for different kind of Ant Colony Optimization based trainers.

By the moment:

  • SingleColACO: A base class for all the single colony ACO-based trainers

  • SingleObjACO: A base class for all the single objective ACO-based trainers

  • ElitistACO: A base class for all the elitist single colony ACO-based trainers

  • WeightedElitistACO: A base class for all the elitist single colony ACO-based trainers where the elite deposit a weighted amount of pheromone

  • PACO: A base class for all the population-based single colony ACO-based trainers

  • AgeBasedPACO: A base class for all the population-based single colony ACO-based trainers with an age-based population update strategy

  • QualityBasedPACO: A base class for all the population-based single colony ACO-based trainers with a quality-based population update strategy
