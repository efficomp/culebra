Each species is evolved in a different subpopulation.

  • pop_sizes (int or Sequence of int, optional) â The population size for each subpopulation (species). If only a single value is provided, the same size will be used for all the subpopulations. Different sizes can be provided in a Sequence. All the sizes must be greater then zero. If set to None, DEFAULT_POP_SIZE will be used. Defaults to None

  • num_subpops (int, optional) â The number of subpopulations (species). If set to None, the number of species evolved by the trainer is will be used, otherwise it must match the number of species. Defaults to None

  • representation_size (int, optional) â Number of representative individuals that will be sent to the other subpopulations. If set to None, DEFAULT_REPRESENTATION_SIZE will be used. Defaults to None

  • representation_selection_func (Callable, optional) â Policy function to choose the representatives from each subpopulation (species). If set to None, DEFAULT_REPRESENTATION_SELECTION_FUNC will be used. Defaults to None

  • representation_selection_func_params (dict, optional) â Parameters to obtain the representatives with the selection policy function. If set to None, DEFAULT_REPRESENTATION_SELECTION_FUNC_PARAMS will be used. Defaults to None

  • checkpoint_enable (bool, optional) â Enable/disable checkpoining. If set to None, DEFAULT_CHECKPOINT_ENABLE will be used. Defaults to None

  • checkpoint_freq (int, optional) â The checkpoint frequency. If set to None, DEFAULT_CHECKPOINT_FREQ will be used. Defaults to None

  • checkpoint_filename (str, optional) â The checkpoint file path. If set to None, DEFAULT_CHECKPOINT_FILENAME will be used. Defaults to None

  • verbose (bool, optional) â The verbosity. If set to None, __debug__ will be used. Defaults to None

property CooperativeTrainer.custom_termination_func: Callable[[Trainer], bool]Â¶

Get and set the custom termination criterion.

The custom termination criterion must be a function which receives the trainer as its unique argument and returns a boolean value, True if the search should terminate or False otherwise.

If more than one arguments are needed to define the termniation condition, functools.partial() can be used:

from functools import partial

def my_crit(trainer, max_iters):
    if trainer.current_iter < max_iters:
        return False
    return True

trainer.custom_termination_func = partial(my_crit, max_iters=10)

Apply the search algorithm.

  • _preprocess_iteration()
