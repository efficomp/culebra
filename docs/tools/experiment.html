Parameters

  • wrapper (Any subclass of Wrapper.) – The wrapper method

  • tr_data (Dataset) – Training data

  • tr_fitness (Any subclass of Fitness) – The fitness used for training

  • tst_data (Dataset, optional) – Test data. If None, tr_data will be used. Defaults to None

  • tst_fitness (Any subclass of Fitness, optional) – The fitness used to test. If None, tr_fitness will be used. Defaults to None.
