class Results¶

Class methods¶

classmethod Results.load(filename: Optional[str] = None) → None¶

Load the results from a backup file.

Parameters:

filename (str, optional.) – File path. If set to None, DEFAULT_RESULTS_BACKUP_FILENAME is used. Defaults to None

classmethod Results.from_csv_files(files: Sequence[str], keys: Optional[Sequence[str]] = None, sep: Optional[str] = None) → None¶

Load some results from several csv files.

Parameters:

  • files (Sequence of str, path objects or file-like objects) – Sequence of files containing the results

  • keys (Sequence of str) – Keys for the different results. One key for each csv file. If omitted, the basename of each file in files (without extension) is used. Defaults to None

  • sep (str, optional) – Separator. If None is provided, DEFAULT_SEP is used. Defaults to None

Raises:

  • TypeError – If sep is not a string

  • ValueError – If files and keys have different lengths

  • ValueError – If any key in keys is not a string

  • FileNotFoundError – If any file in files is not found
