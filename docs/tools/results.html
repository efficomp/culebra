tools.Results class¶

class Results(file: Optional[str] = None, key: Optional[str] = None, sep: Optional[str] = None)¶

Create a results manager.

If a file is provided, the results are loaded from it. If a key is also provided, the results are labeled with it. Otherwise, the results manager keeps empty.

Parameters

  • file (str, path object or file-like objects, optional) – File containing the data

  • key (str, optional) – Key for the data

  • sep (str, optional) – Separator, defaults to DEFAULT_SEP
