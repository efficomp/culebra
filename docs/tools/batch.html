tools.Batch class¶

class Batch(wrapper: Wrapper, test_fitness_function: Optional[FitnessFunction] = None, num_experiments: Optional[int] = None)¶

Generate a batch of experiments.

Parameters:

  • wrapper (Wrapper) – The wrapper method

  • test_fitness_function (FitnessFunction, optional) – The fitness used to test. If None, the training fitness function will be used. Defaults to None.

  • num_experiments (int, optional) – Number of experiments in the batch, defaults to DEFAULT_NUM_EXPERIMENTS
