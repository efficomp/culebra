culebra.tools.Evaluation class¶

class Evaluation(trainer: Trainer, test_fitness_function: FitnessFunction | None = None, results_base_filename: str | None = None)¶

Set a trainer evaluation.

Parameters:

  • trainer (Trainer) – The trainer method

  • test_fitness_function (FitnessFunction, optional) – The fitness used to test. If None, the training fitness function will be used. Defaults to None.

  • results_base_filename (str) – The base filename to save the results.

Raises:

  • TypeError – If trainer is not a valid trainer

  • TypeError – If test_fitness_function is not a valid fitness function

  • TypeError – If results_base_filename is not a valid file name
