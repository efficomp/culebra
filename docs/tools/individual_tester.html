tools.individual_tester¶

Tester for individuals correctness and runtime.

Since individuals are generated randomly, it is necessary to perform tests extensively, with different species parameterizations, in order to check that both the constructor and the breeding operators generate valid individuals, that is, individuals meeting its species constraints.

This tester generate different Species from several values for the number of features and for the proportion of features used to configure the rest of parameteres of the species (see base.species.Species.from_proportion()) and executes the constructor and the breeding operators many times to assert if all the generated individuals meet their species parameters.

The mutation operator is also tested with several mutation probabilities.

List of values for the mutation probability.

Different values for the independent probability for each feature to be mutated. Defaults to DEFAULT_MUT_PROB_VALUES

num_feats_values = [10, 100, 1000, 10000]¶

List of different values for the number of features.

A Species will be generated combining each one of these values for the number of features with each one of the different proportions to test the individual implementation (see base.species.Species.from_proportion()). Defaults to DEFAULT_NUM_FEATS_VALUES

prop_values = [0.0, 0.05, 0.1, 0.15, 0.2, 0.25]¶

List of proportions to generate the different Species.

A Species species will be generated combining each one of these proportions with each one of the different values for the number of features values to test the individual implementation (see base.species.Species.from_proportion()). Defaults to DEFAULT_PROP_VALUES
