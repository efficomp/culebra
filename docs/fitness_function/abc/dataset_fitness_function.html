culebra.fitness_function.abc.DatasetFitnessFunction class¶

class DatasetFitnessFunction(training_data: Dataset, test_data: Dataset | None = None, test_prop: float | None = None)¶

Construct a fitness function.

If test_data are provided, the whole training_data are used to train. Otherwise, if test_prop is provided, training_data are split (stratified) into training and test data. Finally, if both test_data and test_prop are omitted, training_data are also used to test.

Parameters:

  • training_data (Dataset) – The training dataset

  • test_data (Dataset, optional) – The test dataset, defaults to None

  • test_prop (float, optional) – A real value in [0, 1] or None. Defaults to None

Class attributes¶

class culebra.fitness_function.abc.DatasetFitnessFunction.Fitness¶

Handles the values returned by the evaluate() method within a Solution.

This class must be implemented within all the DatasetFitnessFunction subclasses, as a subclass of the Fitness class, to define its three class attributes ( weights, names, and thresholds) according to the fitness function.
