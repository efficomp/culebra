fitness_function.feature_selection module¶

Fitness functions for feature selection.

This module provides the following classes:

  • NumFeats: Dummy single-objective function that minimizes the number of selected features from a Dataset.

  • KappaIndex: Single-objective function that maximizes the Kohen’s Kappa index for classification problems.

  • KappaNumFeats: Bi-objective function composed by the two former functions. It tries to both maximize the Kohen’s Kappa index and minimize the number of features that an individual has selected.
