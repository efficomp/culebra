culebra.fitness_function.tsp module¶

Fitness functions for the TSP problem.

This module is composed by:

  • The abc sub-module, where some abstract base classes are defined to support fitness functions developed in this module

  • Some scoring fitness functions:

    • MultiObjectivePathLength: Multi-objective fitness function for TSP problems.

    • PathLength: Single objective fitness function for TSP problems.
