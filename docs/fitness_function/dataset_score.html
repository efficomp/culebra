culebra.fitness_function.dataset_score module¶

Fitness functions related to dataset scoring.

This module is composed by:

  • The abc sub-module, where some abstract base classes are defined to support fitness functions developed in this module

  • Some popular scoring fitness functions:

    • Accuracy: Single-objective function that maximizes the Accuracy.

    • KappaIndex: Single-objective function that maximizes the Kohen’s Kappa index.

Attributes:¶

DEFAULT_CV_FOLDS = 5¶

Default number of folds for cross-validation.
