culebra.fitness_function.dataset_score.abc module¶

Fitness functions related to dataset scoring.

This sub-module provides several abstract fitness functions to score dataset-related problems:

  • ClassificationScorer: Lets defining dataset classification-related fitness functions

  • DatasetScorer: Allows the definition of dataset-related fitness functions.
