fitness_function.classifier_optimization.RBFSVCFitnessFunction class¶

class RBFSVCFitnessFunction(training_data: culebra.base.dataset.Dataset, test_data: Optional[culebra.base.dataset.Dataset] = None, test_prop: Optional[float] = None, classifier: sklearn.base.ClassifierMixin = SVC())¶

Construct a fitness function.

If test_data are provided, the whole training_data are used to train the classifier. Otherwise, if test_prop is provided, training_data are split (stratified) into training and test data. Finally, if both test_data and test_prop are omitted, training_data are also used to test.

Parameters

  • training_data (Dataset) – The training dataset

  • test_data (Dataset, optional) – The test dataset, defaults to None

  • test_prop (float, optional) – A real value in (0, 1) or None. Defaults to None

  • classifier (SVC with RBF kernels, optional) – The classifier

Class attributes¶

class fitness_function.classifier_optimization.RBFSVCFitnessFunction.Fitness¶

Fitness class to handle the values returned by the evaluate() method within an Individual.

This class must be implemented within all the RBFSVCFitnessFunction subclasses, as a subclass of the Fitness class, to define its three class attributes (weights, names, and thresholds) according to the fitness function.
