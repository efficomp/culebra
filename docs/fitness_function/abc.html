culebra.fitness_function.abc module¶

Abstract base fitness functions.

This sub-module provides several abstract classes that help defining other fitness functions. The following classes are provided:

  • DatasetFitnessFunction: Allows the definition of dataset-related fitness functions.

  • ClassificationFitnessFunction: Lets defining classification-related fitness functions

  • FeatureSelectionFitnessFunction: Designed to support feature selection problems

  • RBFSVCFitnessFunction: Is centered on the hyperparameters optimization of SVM-based classifiers with RBF kernels.
